Descrição Detalhada

Classe abstrata que serve de base para os domínios.

Classe da qual todos os dominios são herdados. Implementa as funçoes setValue e getValue, que servem para definir o valor armazenado em um dominio ou retornar esse valor. Todos os valores para os dominios sao armazenados na string value. É exigida a implementação da função abstrata pura validate em todos os dominios. Essa função define as regras pelas quais valores serao aceitos ou recusados pelo dominio.

Armazena o valor passado, caso este seja valido para o dominio em questão.

A funçao setValue invoca o metodo validate do dominio, com o intuito de verificar se o valor passado é valido. Em caso positivo, o valor é armazenado no campo value do objeto. Em caso negativo, é lançada uma excessão do tipo std::invalid_argument.

Função privada que define as regras de validação para o dominio.

Função pura abstrata, que é sobrescrita nos domínios e define o comportamento de cada domínio. Caso o valor passado nao seja valido, é lançada uma excessão std::invalid_argument.
